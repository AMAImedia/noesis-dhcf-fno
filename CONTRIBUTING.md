# Contributing

NOESIS is a **closed, proprietary product**. This repository is documentation only —
there is no public source code to contribute to.

You're still welcome to engage:

- **Found a doc error or broken link?** Open an issue.
- **Want API access, a partnership, or a press item?** Email **info@amaimedia.com**.
- **Feature request?** Open an issue or email us — see the [ROADMAP](ROADMAP.md).

Pull requests that add code, reverse-engineer the platform, or attempt to reproduce the
technology will be closed. See [LICENSE](LICENSE) and [NOTICE](NOTICE).
