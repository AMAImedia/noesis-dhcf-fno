# Pricing

NOESIS uses **one token pool** (credits) across every tool — no separate per-feature
subscriptions. Subscribe for a monthly credit allotment, or top up any time. Spend the same
credits on dubbing, voice, video, music, or images.

> Live prices and current plans: **[amaimedia.com](https://amaimedia.com)**. Figures below
> are indicative and may change.

## How credits work

**1 credit = $0.001** (so a $50 top-up ≈ 51,000 credits). You're billed per use, only for
what you run. Light tools cost a few credits; the flagship dubbing is priced per minute.
