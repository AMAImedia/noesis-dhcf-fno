# Security & Responsible Disclosure

We take the security and integrity of the NOESIS platform seriously.

## Reporting
Found a security issue in the platform or API? Email **info@amaimedia.com** with details
and steps to reproduce. Please do not open a public issue for security matters.

We aim to acknowledge reports promptly and will coordinate disclosure with you.

## Scope
This repository is documentation only and contains no source code, secrets, or
credentials. Platform and API security reports should reference the live service.

## Content authenticity
NOESIS media may carry provenance signaling so AI-generated or AI-dubbed output can be
identified. Misuse for impersonation or deception is prohibited by our terms.
